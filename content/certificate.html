Detail of Certification MTCNA

MikroTik Certified Network Associate (MTCNA) adalah sertifikasi yang dikeluarkan oleh perusahaan MikroTik untuk membuktikan bahwa seseorang memiliki pengetahuan dasar dan keterampilan dalam mengelola perangkat dan jaringan menggunakan perangkat keras dan perangkat lunak MikroTik.


Berikut ini adalah penjelasan rinci mengenai sertifikasi MTCNA, termasuk materi dan keterampilan yang akan dikuasai:

  1. MikroTik RouterOS Introduction
  2. Configuration DHCP
  3. Configuration Bridging
  4. Configuration Routing
  5. Configuration Wireless
  6. Configuration Firewall
  7. Configuration QoS
  8. Configuration tunnels

Lebih lengkapnya terkait materi MTCNA bisa anda lihat di sini.

Detail of Certification MTCTCE

MikroTik Certified Traffic Control Engineer (MTCTCE) adalah sertifikasi tingkat lanjut yang dikeluarkan oleh MikroTik. Sertifikasi ini menekankan pada pengetahuan dan keterampilan yang diperlukan untuk mengelola lalu lintas jaringan dengan efisien menggunakan perangkat MikroTik RouterOS.

Berikut adalah penjelasan rinci mengenai sertifikasi MTCTCE, termasuk materi dan keterampilan yang akan dikuasai:

  1. Packet Flow Diagram
  2. Firewall Filter, NAT, Mangle
  3. Quality of Service (QoS)
  4. DNS Client, Cache
  5. DHCP Client, Relay, Server
  6. Web Proxy

Lebih lengkapnya terkait materi MTCTCE bisa anda lihat di sini.
