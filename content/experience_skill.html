Pengalaman Kerja

Di bawah ini adalah pengalaman kerja saya selama 6 tahun ini sejak saya lulus dari sekolah kejuruan di SMK Negeri 1 Nglegok.

PT. Solusi Identitas Global Net

Network Engineer L2 (Promosi - Juni 2025 hingga sekarang)

Sebelumnya : Network Administrator L1 (Juni 2024 - Juni 2025)

Lokasi : Sidoarjo | Kontrak - Onsite


Bergabung dengan perusahaan yang bergerak di bidang PSrE (Penyelenggara Sertifikat Elektronik), saya bertanggung jawab dalam membangun dan menjaga infrastruktur jaringan internal dan layanan kripto grafi berbasis PKI (Public Key Infrastructure). Setelah satu tahun menjabat sebagai Network Administrator L1, saya dipercaya mengemban jabatan sebagai Network Engineer L2 untuk memperkuat stabilitas dan keamanan jaringan perusahaan.


Kontribusi & Dampak :

  • Melakukan monitoring dan analisis log akses fisik / non-fisik, termasuk sistem CCTV, perangkat IDS/IPS, serta seluruh perangkat jaringan internal.
  • Mengelola konfigurasi dan segmentasi pada perangkat jaringan seperti FortiGate Firewall, Switch Layer 2 / 3 (H3C, Cisco), dan router MikroTik.
  • Menyusun sistem monitoring lalu lintas jaringan berbasis PRTG dan NetFlow, untuk deteksi dini anomali atau bottleneck.
  • Bertanggung jawab atas dokumentasi, laporan traffic usage, backup rutin konfigurasi, dan prosedur recovery.
  • Menambahkan indikator IOC Malware terbaru ke FortiGate secara manual melalui External Threat Feed Connector.
  • Melakukan riset dan simulasi beberapa teknologi modern seperti ADVPN, Pi-Hole, VXLAN, dan BGP EVPN sebagai Proof-of-Concept untuk peningkatan arsitektur jaringan perusahaan di masa mendatang.

Network Operator Center (Juni 2023 - April 2024)

Lokasi : Remote | Penuh Waktu


Menjadi bagian penting dari fase pengembangan awal ISP lokal ini, saya dipercaya merancang, membangun, dan memelihara jaringan dari nol yang kemudian melayani lebih dari 300 pelanggan di wilayah Lamongan Barat - Selatan.


Kontribusi & Dampak :

  • Merancang jaringan distribusi pelanggan, backbone antar BTS, serta penghubung ke jalur utama UpStream.
  • Mengimplementasikan konfigurasi untuk berbagai perangkat Multi-Vendor seperti MikroTik, Ubiquity, Huawei, HSGQ dan ZTE.
  • Melakukan monitoring performa jaringan End-to-End, termasuk distribusi backbone dan akses pelanggan.
  • Menyusun dan menerapkan manajemen VLAN untuk segmentasi layanan PPPoE dan distribusi IP dinamis pada perangkat pelanggan.
  • Mengoptimalkan kualitas jaringan FTTH dan mengukur redaman optik setiap user modem menggunakan parameter teknis.
  • Melakukan perbaikan struktural pada link wireless Point-to-Multipoint dan sistem distribusi akses RT/RW Net untuk menurunkan angka gangguan pelanggan.

Network Operator Center (Desember 2018 - Mei 2024)

Supervisor Technical Support (Interim) (Juli 2020 - November 2020)

Lokasi : Lamongan Onsite | Penuh Waktu


Selama lebih dari 5 tahun, saya berkontribusi besar terhadap pengembangan jaringan ISP ini, mulai dari fase awal hingga stabil dalam melayani ratusan pelanggan. Selain bertanggung jawab secara teknis, saya juga sempat mengisi jabatan sementara sebagai Supervisor Technical Support.


Kontribusi & Dampak :

  • Menyusun dan mengimplementasikan topologi jaringan LAN dan wireless, termasuk Point-to-Point untuk konektivitas antar BTS dan Point-to-Multipoint untuk konektivitas BTS to User.
  • Mengembangkan dan menerapkan sistem monitoring real-time berbasis alert untuk mendeteksi gangguan perangkat jaringan secara otomatis.
  • Melakukan konfigurasi routing statis dan VLAN ID, serta pemetaan subnet untuk efisiensi distribusi IP.
  • Membangun sistem pemantauan penggunaan bandwidth setiap perangkat jaringan menggunakan SNMP & analisis router.
  • Sebagai Supervisor Interim, saya mengelola jadwal teknisi, menganalisis laporan gangguan, serta menyusun strategi pengembangan jaringan dan pasar berdasarkan data analitik pelanggan.
  • Dalam waktu yang kurang lebih sekitar 6 bulan pada Juli 2020 - Desember 2020, saya juga sempat menjadi Trainer untuk peserta Magang Kerja dari Disnaker di perusahaan ini dengan total peserta 10 orang.

Kemampuan / Keahlian

Di bawah ini adalah beberapa kemampuan / keahlian yang saya telah kuasai, khususnya di beberapa vendor yang sering saya gunakan/konfigurasi.

🛠️ Jaringan & Infrastruktur

  • Perancangan dan implementasi topologi jaringan LAN, wireless PtP/PtMP, serta segmentasi VLAN.
  • Konfigurasi IP Addressing (IPv4 / IPv6), subnetting, ARP, ICMP, dan routing static & dynamic (OSPF, EIGRP).
  • Manajemen switch Layer 2/3 dan routing antar VLAN (Router on a Stick, Inter-VLAN Routing).
  • Instalasi dan konfigurasi access point wireless multi-vendor (TP-Link, MikroTik, Ubiquity, Cisco, Huawei, Ruckus).
  • Implementasi EtherChanel, STP, dan FHRP untuk keandalan jaringan.

🔐 Keamanan Jaringan

  • Konfigurasi firewall (MikroTik & FortiGate) untuk NAT, Filter, Mangle, Raw, dan Layer 7 (L7) inspection.
  • Pemanfaatan Connection State, Mark Routing/Packet, dan Hierarchical Token Bucket (HTB) untuk Quality of Services (QoS).
  • Konfigurasi Access Control Lists (ACL) IPv4, NAT, dan DHCP Snooping.
  • Integrasi IoC (Indicator of Compromise) Malware terbaru ke FortiGate via External Threat Feed.
  • Simulasi dan pengujian teknologi DNS Sinkhole (Pi-Hole), VPN IPSec, ADVPN, BGP EVPN, dan VXLAN.

📡 Monitoring & Observability

  • Penerapan sistem monitoring jaringan menggunakan PRTG, Traffic Flow, dan SNMP.
  • Setup LibreNMS : Instalasi manual / VM / Docker, device discovery via SNMP, konfigurasi MAP & monitoring layanan.
  • Pemantauan kualitas backbone, link distribusi, dan FTTH (Optical Loss & Signal Strength)
  • Troubleshooting jaringan (routing static, wireless interference, downlink / BTS error).

🖥️ Sistem Operasi & Virtualisasi

  • Instalasi dan manajemen OS Windows (client/server) & Linux (Ubuntu, Debian, Kali, Parrot OS).
  • Operasional CLI & GUI, remote desktop, user permission, file & printer sharing.
  • Penanganan VM (vmdk) dan instalasi monitoring di lingkungan virtual.

💻 Version Control & Tools Developer

  • Penguasaan Git dasar hingga intermediate (clone, init, remote, add, commit, branch, merge, revert, restore).
  • Integrasi Github dengan token classic atau GitHub Desktop.
  • Praktik versioning untuk dokumentasi, script backup, dan konfigurasi jaringan.

🔧 Keterampilan Tambahan

  • Instalasi CCTV (Analog, Digital, IP Camera) untuk indoor & outdoor.
  • Manajemen kabel jaringan & kelistrikan dasar (rack server, penomoran kabel, grounding).
  • Penataan perangkat jaringan di rak server dan troubleshooting lapangan.
