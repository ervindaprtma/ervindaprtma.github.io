Kemampuan / Keahlian

Di bawah ini adalah beberapa kemampuan / keahlian yang saya kuasai, khususnya di beberapa vendor yang sering saya gunakan/konfigurasi.

  1. Dasar Jaringan
    • Lapisan dasar OSI/TCP IP.
    • IP Addressing.
    • Subnetting IPv4.
    • Penyusunan topologi jaringan.

  2. MikroTik Network Associate
    • Manajemen user & level hak ases.
    • Konfigurasi firewall dasar (Filter, Mangle, NAT).
    • Konfigurasi bridge.
    • Konfigurasi static routing.
    • Konfigurasi tunnel (EoIP, PPTP, PPPoE).
    • Konfigurasi manajemen QoS (Quality of Services) seperti Simple Queue.
    • Konfigurasi wireless mode (AP Bridge, Bridge, Station, Station-Bridge).
    • Konfigurasi LoadBalance dengan metode PCC & ECMP
    • Konfigurasi manajemen VLAN (Virtual LAN).

  3. MikroTik Traffic Control Engineer
    • Dasar-dasar keamanan jaringan.
    • Konfigurasi firewall lanjutan (Filter, Mangle, NAT, Raw).
    • Analisa & pemanfaatan Connection State untuk melakukan konfigurasi firewall.
    • Konfigurasi firewall layer 7.
    • Konfigurasi manajemen QoS lanjutan (Simple Queue & Queue Tree) dikombinasikan dengan Firewall Mangle (Mark Routing/Packet) dan menggunakan sistem parent.
    • Perhitungan HTB (Hierarki Token Bucket).

  4. MikroTik Routing Engineer
    • Dasar-dasar routing.
    • Konfigurasi routing static router MikroTik.
    • Konfigurasi routing dynamic OSPF.
    • Konfigurasi routing OSPF Multi-area.
    • Konfigurasi routing OSPF redistribute.

  5. Cisco Network Associate
    • Memahami dasar jaringan seperti lapisan OSI/TCP IP.
    • Memahami perbedaan konfigurasi perangkat router dan switch lapisan 2/3.
    • Konfigurasi VLAN ID di switch lapisan 2/3.
    • Konfigurasi dasar router cisco.
    • Konfigurasi router on stick (gateway per VLAN ID).
    • Konfigurasi routing static.
    • Konfigurasi routing dynamic OSPF & EIGRP.
    • Konfigurasi Access List Allow/Deny koneksi per IP dan per port service.
    • Konfigurasi NAT di router cisco.
    • Backup dan restore konfigurasi.

  6. Jaringan Wireless
    • Dasar jaringan wireless (protokol 802.11-A/B/G/N/AC).
    • Merancang topologi jraingan wireless PtP (Point-to-Point) dan PtMp (Point-to-Multipoint).
    • Konfigurasi jaringan wireless di beberapa vendor (Tp-Link, MikroTik, Ubiquity, Cisco, Tenda, Netis, Ruckus, Asus, dan lainnya).

  7. Sistem Operasi
    • Instalasi sistem operasi Windows (termasuk windows server) dan Linux (Ubuntu, Debian, Kali-Linux, Parrot OS).
    • Mengoperasikan sistem operasi Windows dan Linux melalui CLI (Command Line Interface) maupun GUI (Graphical User Interface).
    • Manajemen sharing file & printer dari windows.
    • Konfigurasi remote desktop untuk windows.
    • Manajemen user & hak akses file/folder.

  8. Git dan GitHUb
    • Git clone repository.
    • Git init.
    • Git remote untuk suatu repository menggunakan account github desktop atau token clasic.
    • Git push repository lokal ke repository github.
    • Git pull & git fetch untuk update repository lokal dari repository github.
    • Git add & git commit.
    • Git reset & git revert.
    • Git restore
    • Git branch & git merge.

  9. Sistem Monitoring Jaringan LibreNMS
    • Instalasi aplikasi LibreNMS dengan VM.vmdk atau manual instalasi atau menggunakan docker.
    • Melakukan konfigurasi Globe dan dashboard di LibreNMS.
    • Menambahkan perangkat menggunakan SNMP.
    • Melakukan konfigurasi MAP Location & Group perangkat.
    • Melakukan konfigurasi monitoring service tertentu serta aplikasi service di perangkat yang telah ditambahkan.

  10. Pengetahuan Dasar Teknis
    • Instalasi CCTV indoor & outdoor baik NVR, IP Camera, dan DVR.
    • Manajemen kabel jaringan & listrik dasar.
    • Manajemen rak server.
