Project BTS
Detail of Project Wireless Distribution Outdoor di 22 BTS dengan MikroTik & Air Fiber

Project ini sebenarnya dimulai secara bertahap, namun secara keseluruhan dapat disimpulkan seperti berikut ini:

Tujuan: Membangun sebuah jaringan berskala MAN sebagai jalur distribusi jaringan ke setiap rumah di desa-desa pada wilayah Kabupaten Lamongan.

Langkah - langkah implementasi:

  1. Menentukan area yang ditargetkan
  2. Melakukan site survey dan menganalisa area berdasarkan jarak, ketinggian tanah (kontur tanah), dan target bandwidth yang akan dikirimkan
  3. Melakukan survey lokasi yang akan digunakan sebagai titik Base Station (Tower/Antena BTS)
  4. Melakukan analisa link planner dari Tower Pusat menuju ke setiap site menggunakan titik koordinat BTS sebagai acuan lokasi antena
  5. Melakukan analisa coverage area menggunakan titik koordinat BTS sebagai acuan penyebaran
  6. Menentukan jenis antena yang digunakan sebagai PtP dan PtMP
  7. Instalasi perangkat di setiap tower baik pusat dan BTS
  8. Melakukan konfigurasi perangkat dan instalasi wireless link (pointing antena, config frequency, dan lain-lain)
  9. Melakukan tunning pada setiap link menuju ke BTS dengan acuan target bandwidth serta kualitas link
  10. Melakukan pengujian koneksi PtP
  11. Melakukan konfigurasi pada perangkat distribusi (AP) di setiap BTS
  12. Melakukan instalasi pada setiap client (home based) dengan station radio
  13. Melakukan tunning pada setiap link user
  14. Melakukan pengujian koneksi PtMP
  15. Review dan dokumentasi
  16. Monitoring & troubleshooting apabila ditemukan gangguan pada koneksi baik backbone & user

Tools yang digunakan untuk Link Planner:

  1. Conota Camera: Sebagai photo sekaligus coordinate location
  2. UISP Design Center: Sebagai link planner dan penentuan spesifikasi perangkat
  3. Google Earth Pro: Sebagai link planner dan pengecekan kontur tanah
Project The Dude
Detail of Project Monitoring Jaringan dengan The Dude MikroTik

Project ini dibuat atas dasar dibutuhkannya sebuah sistem monitoring namun berjalan secara lokal dengan detail seperti berikut ini:

Tujuan: Membangun sebuah sistem monitoring untuk seluruh jaringan yang telah dibangun menggunakan beberapa protokol seperti ICMP & SNMP.

Langkah - langkah implementasi:

  1. Melakukan instalasi VMware Workstation
  2. Melakukan instalasi CHR MikroTik
  3. Aktivasi Lisensi MikroTik dengan level P1
  4. Instalasi package The Dude
  5. Proses pembuatan Map Network pada The Dude
  6. Menambahkan device baru pada The Dude untuk dimonitoring
  7. Proses pembuatan mapping device pada Map Network yang telah dibuat
  8. Manajemen service yang dimonitoring
  9. Penambahan integrasi SNMP & OID MikroTik
  10. Manajemen Map Network agar lebih terstruktur dengan Device Tree

Tools yang digunakan untuk Install Monitoring System :

  1. VMware Workstation 17 : Sebagai tempat instalasi Virtual Router MikroTik
  2. File CHR MikroTik : Sebagai bahan instalasi Virtual Router MikroTik
  3. Package Installer The Dude : Sebagai service package yang wajib diinstall ke CHR MikroTik
